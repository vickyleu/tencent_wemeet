Tidak boleh membuat sambungan kepada Internet.
Rangkaian selular atau Wi-Fi pada peranti anda mungkin telah dimatikan.

Jika ya, anda boleh cuba langkah-langkah berikut untuk membuat sambungan kepada Internet:

Pilih dan sambung kepada rangkaian Wi-Fi yang tersedia dalam Tetapan > Wi-Fi pada peranti anda.

Dayakan data selular dalam Tetapan > Selular pada peranti anda (pembawa anda mungkin mengecaj untuk penggunaan data selular apabila ciri ini didayakan).

Jika anda disambungkan kepada rangkaian Wi-Fi:

Sila semak sama ada rangkaian Wi-Fi yang disambungkan itu mendapat sambungan kepada Internet, atau sama ada rangkaian tersebut membolehkan peranti anda mengakses Internet.
