인터넷에 연결하지 못했습니다
장치가 모바일 또는 Wi-Fi 네트워크를 사용하지 않습니다

인터넷에 연결해야 하는 경우 다음 방법을 참조하세요.

장치의 "설정"-"Wi-Fi 네트워크" 설정 패널에서 사용 가능한 Wi-Fi 핫스팟을 찾아 선택합니다.

장치의 "설정"-"네트워크" 설정 패널에서 셀룰러 데이터를 활성화합니다(활성화 후 통신사의 데이터 통신 비용을 부과할 수 있음).

Wi-Fi 네트워크에 액세스한 경우

연결된 Wi-Fi 핫스팟이 인터넷에 이미 액세스했는지, 또는 해당 핫스팟이 장치의 인터넷 액세스를 허용했는지 확인하세요.
