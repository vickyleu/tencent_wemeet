Unable to connect to the internet.
The cellular or Wi-Fi network on your device may have been turned off.

If so, you can try the following methods to connect to the internet:

Select and connect to an available Wi-Fi network in Settings > Wi-Fi on your device.

Enable cellular data in Settings > Cellular on your device (your carrier may charge for cellular data use when it is enabled).

If you are connected to a Wi-Fi network:

Please check whether the Wi-Fi network you are connected to is connected to the internet, or whether the network allows your device to access the internet.
