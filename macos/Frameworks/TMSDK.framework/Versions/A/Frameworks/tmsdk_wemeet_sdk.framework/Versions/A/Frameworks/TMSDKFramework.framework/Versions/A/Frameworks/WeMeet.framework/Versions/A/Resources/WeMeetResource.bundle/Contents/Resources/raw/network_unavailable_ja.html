インターネットに接続できませんでした
あなたのデバイスではモバイルネットワークまたはWi-Fiネットワークが有効になっていません

インターネットに接続する必要がある場合は、次の方法を参照できます

デバイスの「設定」-「Wi-Fiネットワーク」設定パネルから、利用可能なWi-Fiホットスポットを選択して接続します。

デバイスの「設定」-「ネットワーク」設定パネルで、セルラーデータを有効にします（有効にすると、キャリアがデータ通信料を徴収する場合があります）。

Wi-Fiネットワークに接続している場合：

接続しているWi-Fiホットスポットがインターネットにアクセスしているか、またはそのホットスポットがあなたのデバイスにインターネットへのアクセスを許可したかを確認してください。
